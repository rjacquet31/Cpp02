# C++ Module 02

> **42 cursus** · C++ · object-oriented programming module.

Part of the C++ learning modules at 42. Ad-hoc polymorphism, operator overloading and the orthodox canonical class form (a fixed-point number class).

## Features

- Multiple exercises (`ex00`, `ex01`, …), each in its own folder
- Compiled with `c++ -Wall -Wextra -Werror -std=c++98`

## Build

```bash
cd ex00 && make    # build a single exercise
```

## Usage

```bash
./<exercise_executable>
```

## Key concepts

Operator overloading and canonical class form.
